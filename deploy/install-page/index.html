Offline Capture Assistant

离线专网页面采集工具

v0.1.0
下载扩展包

解压下载的 ZIP 文件

将 offline-capture.zip 解压到一个固定位置，比如 D:\extensions\offline-capture

注意：安装后不要删除或移动这个文件夹

打开扩展管理页

在地址栏输入 edge://extensions 并回车

开启开发人员模式

点击页面左下角的「开发人员模式」开关

加载扩展

点击「加载解压缩的扩展」按钮，选择刚才解压的文件夹

开始使用

工具栏出现蓝色图标 → 点击打开侧栏 → 在页面上右键「采集此元素」

升级方法：重新下载最新 ZIP，覆盖解压到同一文件夹，然后在扩展管理页点击「重新加载」按钮即可。不需要重新安装。
